¿Como puedo agendar una cita?

Tienes que registrarte como un usuario dentro de nuestra pagina y seguidamente llenar el formulario de Agendar Cita, luego te enviaremos un correo de confirmacion con la fecha de tu cita. Esta fecha puede ser cambiada de acuerdo a la disponibilidad.

¿Como se que tipo de mantenimiento necesito?

El mantenimiento correctivo: es la actividad técnica ejecutada cuando sucede una avería y tiene como objetivo, restaurar el activo para dejarlo en condiciones de que pueda funcionar como se pretende ya sea con su reparación o sustitución. El mantenimiento preventivo: se trata de un tipo de mantenimiento planificado que se realiza incluso cuando un equipo mantiene su capacidad operativa.

Politicas de Privacidad

Mantenimientos Don Saito se preocupa por la proteccion de datos de caracter personal de sus usuarios y clientes y el ejercicio de los derechos que contempla la ley sobre proteccion de datos. De esta forma, el usuario podra siempre solicitar a Mantenimientos Don Saito la actualizacion, modificacion, eliminacion o bloqueo de sus datos personales.

Este sitio web protege automaticamente toda informacion transmitida desde el computador del usuario a sus servidores utilizando el protocolo secure sockets layer (SSL). La informacion que llega al sitio web, es almacenda en servidores de un centro de datos que esta altamente protegido tanto fisica como electronicamente.

Los datos personales y de navegacion que recolecta Mantenimientos Don Saito son usados para procesar las solicitudes y transacciones del cliente, asi como tambien para proveer servicios de maxima calidad y enterder las necesidades y el comportamiento de los usuarios.
